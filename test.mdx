## Setup Resend

<Steps>
  <Step title="Create API Key">
    1. Head over to [Resend Onboarding](https://resend.com/onboarding)
    2. Click **Add API Key**
  </Step>
  <Step title="Verify your domain">
    On the same page, scroll down and click **Add domain**

    **or **go to this page [**https://resend.com/domains**](https://resend.com/domains)

    Once you complete, click **Verify DNS Record**
  </Step>
  <Step title="Audience ID (optional)">
    Go to [https://resend.com/audiences](https://resend.com/audiences/97a201bc-5f10-47ff-b88b-c6ca7ce5ed3b), you can find the Audience ID

    <Note>
      This is used for the **waitlist** and **newsletter** feature as well as **email marketing**.
    </Note>
  </Step>
  <Step title="Add the Keys">
    Add the following keys to your `.env` file

    ```bash
     RESEND_API_KEY=
     RESEND_AUDIENCE_ID=
     FROM_EMAIL=
    ```
  </Step>
</Steps>
